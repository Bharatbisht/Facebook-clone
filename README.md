# Facebook-clone
in this project i used php(pdo + oops). code is easy to read.

Create a database with the name of facebook.
Create Three table with the name of fbtable,admin,post.
Create 8 column LoginId(Primary),Name,Password,Gender,Email,DOB,Address,Image in fbtable.
Create 2 column Name & Password in admin.
Create 2 column LoginId & Post in post.
and run index.php
